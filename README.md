# alura-tracker

Projeto de aplicação de tracker de tarefas utilizando o framework front-end Vue.js 3 

### Deploy no Vercel

https://alura-tracker-wheat-three.vercel.app/

### Tecnologias Utilizadas

- Vue.js 3
- Vuex
- Vue Router
- Axios
- FortAwesome Icons
- Bulma
- JSON Server
